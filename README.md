* Сравнение столбцов в Microsoft Excel для поиска строк с одинаковыми или разными значениями: <br />

  _Шаг 1._ Для сравнения необходимо выделить  все содержимое двух столбцов без заголовков. Значения в ячейках могут быть как текстовыми, так и числовыми или даже смешанными.<br />
  _Шаг 2._ Далее нажать сочетание клавиш Ctrl + G для вызова меню настройки выделения.<br />
  _Шаг 3._ Откроется окно, в котором необходимо нажать на кнопку «‎Выделить».<br />
  _Шаг 4._ В отобразившемся окне выбрать значение «‎отличия по строкам» и нажать на кнопку ОК.<br />
  _Шаг 5._ Таким образом, строки в двух столбцах с уникальными значениями будут выделены. Т.к. данное выделение может быть случайно сброшенно, необходимо перейти на панель инструментов Excel и выбрать любой цвет заливки.<br />
  _Шаг 6._ После заливки выделенных ячеек все уникальные значения в правом столбце будут выделены отдельным цветом, а одинаковые – останутся без заливки.<br />
