* Сравнение столбцов в Microsoft Excel для поиска строк с одинаковыми или разными значениями
  1) Для сравнения необходимо выделить  все содержимое двух столбцов без заголовков. Значения в ячейках могут быть как текстовыми, так и числовыми или даже смешанными.
  2) Далее нажать сочетание клавиш Ctrl + G для вызова меню настройки выделения.
  3) Откроется окно, в котором необходимо нажать на кнопку «‎Выделить».
  4) В отобразившемся окне выбрать значение «‎отличия по строкам» и нажать на кнопку ОК.
  5) Таким образом, строки в двух столбцах с уникальными значениями будут выделены. Т.к. данное выделение может быть случайно сброшенно, необходимо перейти на панель инструментов Excel и выбрать любой цвет заливки.
  6) После заливки выделенных ячеек все уникальные значения в правом столбце будут выделены отдельным цветом, а одинаковые – останутся без заливки.
